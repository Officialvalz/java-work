# java-work

Java practice and small exercises repository.

## Structure

- Root-level `.java` files: standalone exercises and classwork
- `src/main/java/Stack.java`: stack implementation
- `src/test/java/StackTest.java`: JUnit 5 tests for `Stack`

## Requirements

- Java 17
- Maven
- Git

## Run Tests

```bash
mvn test
```

## Notes

Compiled `.class` files and IntelliJ project files are ignored so the repository stays focused on source code.
